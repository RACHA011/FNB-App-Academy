# Calculator #fnb-academy

This is just a simple calculator application that can perform basic mathematical operations like addition, subtraction, multiplication, and division.
 
## Know issues 
- the you can not use nagative numbers, but the calculator can handle nagative numbers 
- you cant press multiple operants at the same time
